# SEA Store-Ready Docs

Este pacote organiza o plano para levar o SEA para Web + Android + iOS com qualidade de loja.

## Arquivos

- `01-device-matrix.md`: matriz de dispositivos, breakpoints e criterios de aceite.
- `02-store-release-checklist.md`: checklist de release para produto, qualidade e lojas.
- `03-capacitor-plan.md`: plano de empacotamento nativo com Capacitor para Next.js.

## Ordem sugerida de execucao

1. Fechar matriz de dispositivos alvo.
2. Rodar checklist de qualidade por prioridade.
3. Iniciar setup Capacitor e ciclo de testes nativos.

## Meta

Entregar uma versao unica e consistente do SEA em celular, tablet e desktop, com caminho claro para Play Store e App Store.
